		テーマ
                ミニゲームサイト

		制作の目的（なぜそのテーマに決めたか）
　　　　　　　　何故、私がこのテーマにしたのかそれはですね。
　　　　　　　　授業で習った事を生かして、サイトを作ってみたかったからです。
　　　　　　　　一番サイトを作るうえで手っ取り早いと思ったのが、
　　　　　　　　ブラウザゲームです。
　　　　　　　　ですが、
　　　　　　　　既にあるブラウザゲームだと需要がないと感じたので
　　　　　　　　ミニゲームがいっぱい遊べるまとめサイトを作ろうと
　　　　　　　　決めました。
　　　　　　　　また、いままで触ったことがなかった、javascriptのライブラリやphpを
　　　　　　　　この機会に触って使えるようになって、今後の役に立つようにサイト作りの勉強も
　　　　　　　　かねて、卒業制作をしました。

		使用言語
　　　　　　　　主に5つです。
　　　　　　　　C# html css javascript php です。
　　　　　　　　C#はサイトとは関係ないですが、
　　　　　　　　今回、UnityのWebGLを使って、
　　　　　　　　Unityで作ったゲームをwebで遊べるようにしています。
　　　　　　　　C#はUnityでゲーム開発をするときに使いました。

		使用環境（開発のために必要な環境）
                windows10
                テキストエディタはvscodeを使いました。
　　　　　　　　また、ゲームエンジンはUnityを使いました。
                以上が主な制作目的・使用言語・使用環境紹介でした。
                次に成果物の名称と成果物の概要を紹介します。
　　　　　　　　そのあと、成果物を実際に見せたいと思います。

		成果物の名称
　　　　　　　　ミニゲームサイトです。
　　　　　　　　名前に深い意味はないです。

		成果物の概要
			どのような制作物か
　　　　　　　　　　　　いろんなミニゲームが遊べるサイトになっています。
			画面遷移がある場合はその内容
　　　　　　　　　　　　レスポンシブ対応していて、
　　　　　　　　　　　　スマホ画面になると上のヘッダーがハンバーガーメニューになったり
　　　　　　　　　　　　縦に表示されるようになっています。
			操作方法　
　　　　　　　　　　　　PCの場合はクリック・キーボード
　　　　　　　　　　　　スマホの場合はタップ操作です
　　　　　　　　　　　　また、ゲームの内容によってはスマホでは遊べないものもあります。
　　　　　　　　　　　　例えば、タイピングゲームなどが、スマホでは、できません。
　　　　　　　　　　　　ゲームは一人モードしかありません。
　　　　　　　　　　　　対戦ゲームではなく、スコアリングゲームとなっております。
